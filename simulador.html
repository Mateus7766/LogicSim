Informações De Uso.

Como usar o circuito
Dentro da interface tem vários botões onde tem as entradas codinome input as saídas codinome output tem o deletar onde é usado para deletar as portas,  entradas,  saídas e fios. Também tem o botão de limpar onde faz a mesma função de deletar só que ele limpar todo o simulador.
Na mesma interface temos os botões das portas com 7 portas disponíveis no simulador,  Leia as informações das portas abaixo para entender a lógica de cada porta.
Como usar a Porta AND (E)
Só retorna 1 se todas as entradas forem 1.
Como usar a Porta OR (OU)
Só Retorna 1 se pelo menos UMA entrada for verdadeira.
Como usar a Porta NOT (NÃO)
Inverte o valor, irá trocar o 1 por 0 ou 0 por 1.
Como usar a Porta NAND (NOT AND)
O NAND é o negado do AND onde se você coloca 0 o resultado vai sair 1 pois o 0 vira 1,   Se você coloca 0 e 1 a saída será 1 pois o 0 vira negado mas ser coloca 1 e 1 a saída será 0,   Se houver qualquer zero na jogada, a saída será 1.
Como usar a Porta NOR (NOT OR)
A porta NOR é o OR negado. Se definirmos as entradas como 0 e 0,  a saída será 1. Se definirmos as entradas como 0 e 1,  a saída será 0;  o mesmo ocorrerá se as entradas forem 1 e 1.
Como usar a Porta XOR (OR Exclusivo)
A porta XOR (Ou Exclusivo) retorna 1 sempre que a quantidade de entradas em nível alto (1) for ÍMPAR. Se a quantidade de entradas "1" for par (como 0, 2 ou 4), a saída será 0. Em sistemas simples com apenas duas entradas, ela retorna 1 apenas quando os valores são diferentes.
Como usar a Porta do XNOR (OR NOT Exclusivo)
O XNOR possui duas regras. A primeira é a da **igualdade**: se todas as entradas forem iguais (tudo 0 ou tudo 1), a saída será 1. A segunda é a da **paridade** para três ou mais entradas: se a soma das entradas "1" for par (ex: 1+1=2 ou 0+0=0), a saída será 1. Se a soma for ímpar (ex: 0+0+1=1 ou 1+1+1=3), a saída será 0.
